# Frequently Asked Questions

### Can I deploy this application on Render?
Yes, you can deploy the React app as a static site and the Node/Express backend as a web service.

### Where does the price matrix come from?
The database is seeded using data from legacy price spreadsheets via the `seed.ts` script.

### What is SSE?
Server-Sent Events (SSE) is a web technology that allows a server to stream real-time updates to a browser client over a single HTTP connection.
