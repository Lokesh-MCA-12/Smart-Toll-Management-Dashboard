# System Features and Capabilities

## Core Features

### 1. Live Lane Simulator
- Simulates RFID transits across toll segments (e.g. Mindanao Ave, Valenzuela, Bocaue).
- Validates account balances and triggers balance warnings in real-time.

### 2. Dynamic Rates Matrix Controller
- Allows operators to update segment pricing for Class 1, Class 2, and Class 3 vehicles.
- Propagates rate changes to active lanes in real-time.

### 3. Server-Sent Events (SSE) Telemetry
- Streams passage events to the dashboard.
- Updates tables and charts instantly without page refreshes.

### 4. Interactive Analytical Charts
- Visualizes traffic volumes, busy plazas, and revenue splits using Recharts.
