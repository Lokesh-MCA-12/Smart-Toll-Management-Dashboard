# 🎫 Smart Toll Management Dashboard System

![System Preview](../Demo%20Video-Smart%20Toll%20Management%20Dashboard.mp4)

A real-time, modern full-stack web application designed for smart highway toll plazas. This system modernizes legacy Java Swing tolling setups into a cloud-ready operational dashboard featuring live RFID entry/exit gate simulation, dynamic pricing matrices, and real-time passage telemetry.

## 🛠 Tech Stack Overview
* **Frontend Web App**: React, TypeScript, Vite, Tailwind CSS, Lucide Icons, Recharts
* **Backend REST API**: Node.js, Express, TypeScript, Server-Sent Events (SSE)
* **Database & ORM**: SQLite, Prisma ORM
* **Legacy Modules**: Java Swing Desktop App (provided for reference), Python Excel Hash Parsers
